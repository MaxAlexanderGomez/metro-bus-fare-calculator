
# Metro Bus Fare Calculator

A Python command-line program that calculates the total number of passengers carried by three buses, the total fare earned, and the remaining net profit after operating costs.

## Features

- Accepts passenger totals for three buses
- Calculates the combined number of passengers
- Calculates fare revenue at $2.50 per passenger
- Deducts 50% for operating costs
- Uses functions, arguments, and return values

## Python Concepts Used

- Functions
- Function parameters
- Return values
- Integer input
- Arithmetic operations
- String conversion
- Formatted output
